# neovim-practise-101-分割合并行

## Origin text

```
<leader>tt <leader>tr <leader>tl <leader>ts <leader>tT
```

## 根据空格分割成多行后

```
<leader>tt
<leader>tr
<leader>tl
<leader>ts
<leader>tT
```

## 合并成带空格的多行后

```
<leader>tt <leader>tr <leader>tl <leader>ts <leader>tT
```

## 合并成不带空格的多行后

```
<leader>tt<leader>tr<leader>tl<leader>ts<leader>tT
```
